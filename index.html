Description

I had the unique opportunity to create a website for CTU Main's Scholarship office, as my wife works in the office, and her boss expressed a need for a website for the office. I was able to communicate with them and understand their needs for the website. Leveraging my skills in HTML, CSS, JavaScript, and Bootstrap, I crafted an intuitive and visually appealing website that serves as a centralized hub for scholarship-related information.

Client's requirements
  • Displays updates and announcements from their facebook page
  • Provides Information about scholarships
  • Provides Contact Information, as well as a direct way to contact the team
  • Visually Attractive
My Role: Solo Website Developer
Technologies Used
  • Facebook Graph API to ensure that students are always in the loop about the latest announcements and happenings.
  • ChatGPT API to enable students to ask questions and receive AI-driven responses, streamlining the communication process and offering quick solutions to common queries.
  • HTML
  • CSS
  • Bootstrap for responsive design
  • JavaScript
  • Node.js for security with API keys
